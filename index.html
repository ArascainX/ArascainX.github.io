Co jsem se naučil

Během svých kurzů jsem získal pevné základy a praktické dovednosti v několika klíčových oblastech webového vývoje. Naučil jsem se programovat v Pythonu s důrazem na OOP, pracovat s knihovnami Pandas a NumPy pro datovou analýzu, vytvářet webové aplikace pomocí frameworku Django a jeho rozšíření Django REST API. Dále jsem si osvojil frontendové technologie, včetně Reactu, JavaScriptu, HTML5 a CSS, a zvládnul jsem správu verzí pomocí Git a práci s databázemi SQLite. Tyto znalosti mi umožňují komplexně vyvíjet moderní webové aplikace a řešit různé programátorské výzvy.
